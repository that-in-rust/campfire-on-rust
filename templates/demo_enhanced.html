🚀 Quick Start Guide

  1. Choose any demo user from the credentials below (all passwords are "password")
  2. Click a credential card to auto-select that user for login
  3. Explore pre-loaded rooms: General, Development, Design, Product Planning, and more
  4. Try interactive features: @mentions, /play sounds, full-text search across all messages
  5. Multi-user simulation: Open multiple browser tabs to log in as different users and see real-time chat
  6. Test enterprise features: Room permissions, bot integration, push notifications
💡 Pro Tip for Realistic Testing:
  • Open 2-3 browser tabs and log in as different users (Alice, Bob, Carol)
  • Start a conversation in one tab and watch it appear in real-time in the others
  • Try @mentioning users: @alice or @bob
  • Play sounds with commands like: /play tada, /play yeah, /play nyan
  • Use the search feature to find old messages across all rooms
